 以下を全てお読みになってから、このソフトウェアをご利用下さい。
------------------------------------------------------------------------------

　聞々ハヤえもんをダウンロードしていただき、ありがとうございます。

　このソフトは、WAVE, MP3, Ogg Vorbis, WMA, M3U などの各種フォーマットに対応し
た、再生速度／再生周波数／音程を自由に変更できる MP3 プレーヤとして公開中のオー
プンソースのフリーソフトです。

　このソフトに関するご意見・ご感想、開発に参加してくださる方などは大歓迎です。
ぜひTwitter（@ryota_yama）やメールなどでお知らせ下さい。

------------------------------------------------------------------------------
 動作環境
------------------------------------------------------------------------------

　Windows 10 以降の OS に対応しています。動作報告は歓迎します。
　DirectX 3 以降が導入されている環境が必要です。

------------------------------------------------------------------------------
 開発環境
------------------------------------------------------------------------------

　Windows 10 / Visual Studio Community 2017, Windows SDK を使い、開発を行って
います。

　サウンド関連の実装には、高性能オーディオライブラリ「BASS」を使用しています。
有用なライブラリを公開なさっている Ian Luck 氏に、深く感謝致します。

	Un4seen Developments : http://www.un4seen.com/

------------------------------------------------------------------------------
 インストール
------------------------------------------------------------------------------

　インストーラを利用していただくか、ZIPを適当な場所に解凍してください。

　アンインストールする時は、コントロールパネルのプログラムと機能からアンインス
トールしていただくか、
　解凍して配置したファイルを全て削除して下されば結構です。

　このソフトは、レジストリへの書き込みは行っていません。

 マニュアル
------------------------------------------------------------------------------
