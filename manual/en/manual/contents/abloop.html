●Using Marker play

With this software, you can use "Marker play" function to loop playback of the interval. Looping playback of the interval is useful to learn playing the guitar, piano, etc.

Let's using this function.

While playing audio file, by clicking the button of toolbar that says "Marker play", you can turn on this function. The time slider changes to "Marker play" mode.

▼Marker play mode

Where the slider is displayed in blue is loop interval.

At this time, By clicking the button of toolbar that says "Add a marker", you can add a marker and separate the interval.
It is possible to add a marker more than once.

▼Add a marker

To Change the loop section, click on the section you want to loop.
You can also change it by clicking button "Head/Previous marker" or "Next marker".

▼Change the loop section

To change a marker position, drag a marker to the left or right.

▼Change a marker position

If you want to cancel "Marker play", click the button "Marker play".

▼Cancel "Marker play"
