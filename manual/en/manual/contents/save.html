●Save files

With this software, you can save the audio file you change the speed frequency, pitch, etc.

Save target is the following two pattern.

<Current song>
To save current song, select "File" -> "Save" menu.
And input the file name and select file format.

<All songs in playlist>
To save all songs in playlist, select "File" -> "Save all" menu and select file format.
*File name is "Original file name" and "_h001". "001" part is sequencial number starting from "001".
